Theorem (p. 13): A set is linearly dependent if and only if one vector in the set is in the span of the other vectors in the set.

Characterizing dependence: Tossing in 0⃗\vec{0}0 will always result in a dependent set.

Theorem (p. 16): If a set of vectors {v1⃗,⋯ ,vk⃗}\{\vec{v_1}, \cdots , \vec{v_k}\}{v1​​,⋯,vk​​} contains the zero vector then the set is linearly dependent.
